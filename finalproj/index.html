CS 184: Computer Graphics and Imaging, Spring 2022

Final Project: Adjustable Liquid Simulation

Amy Jiang, Christopher Thomas, Jeremy Ahn, Jim Kang

2D simulation for various liquids with differing properties that may result in different fluid movement/ interaction with solid objects in simulation. Able to observe how the drop of solid mass affects the fluid movement of different liquids.

Problem Description

The problem we are trying to solve is how one can replicate/simulate the fluid movement of different liquids with severally differing properties in computer graphics. For example, while liquids such as water follow Newtonian properties, other liquids such as honey do not. We want to not only be able to simulate their fluid movement, but also simulate how each liquid will interact with a solid mass. The implementation of the fluids will be difficult, but we imagine that implementation of an accurate interaction between fluids and solid mass will be the core difficulty of our project.

At the moment, we're not sure how to measure the quality of work aside from "it looks accurate" or reasonable compared to expected behavior of the simulated liquids.

We plan to answer if traditional Navier-Stokes equations can be used for non-Newtonian fluid and if so, what changes we can make. We also want to explore and answer how fluid interacts with solid mass and how our equations will change based on this interaction.

  1. Plan
    1. Create a liquid simulation in which a user can choose a solid geometry to drop onto a liquid with adjustable viscosity (eg. can simulate water, with low viscosity, or honey, with high viscosity).
    2. The final result will include a video result of the interaction between the selected solid and liquid.
    3. The performance of the system will be evaluated based on the quality of the simulation in regards to real life physics projected onto a 2D plane.
    4. Want to answer how we can represent the physics of liquids using computer technology.
  2. Hopes
    1. Add support for non-Newtonian fluids, such as oobleck (shear thickening) or mayonaise (shear thinning).
    2. The final hope is to possibly create a 3D interactive extension with more complex fluids and thus fluid dynamics.
    3. The interactive demo will be a run through of the different liquids and moving around the solid within the liquid.
    4. The performance of this system will be similar to that of the 2D simulation but should allow for different physics equations.
